Display Flex

The display: flex property in CSS allows you to create flexible layouts in a container. Alongside display: flex, there are several other properties that work with it to control the layout of flex items.

Flex Properties

display: This is the primary property to enable flexbox layout. It can take the following values.

flex-direction: This property defines the direction in which the flex items are placed within the flex container. It can take the following values.

justify-content: This property aligns flex items along the main axis of the flex container. It can take the following values.

align-items: This property aligns flex items along the cross axis of the flex container. It can take the following values.

align-self: This property allows the alignment of individual flex items along the cross axis, overriding the align-items property of the flex container.

flex-wrap: This property controls whether the flex container is single-line or multi-line. It can take the following values.

flex-grow, flex-shrink, and flex-basis: These properties can be used together as the flex shorthand property to specify how a flex item will grow, shrink, and the initial size of the flex item.

order: This property allows you to control the order in which flex items appear within their flex container. By default, all items have a value of 0.
